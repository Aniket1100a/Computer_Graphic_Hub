Understanding Transformations in Computer Graphics

What are Transformations?

Transformations in computer graphics refer to the process of changing the position, size, or orientation of objects in a 2D or 3D space. Transformations are fundamental operations used to create various effects and animations in graphics applications.


Translation

Translation moves an object from one location to another by a fixed distance along a specified direction. It shifts the object horizontally and vertically in 2D graphics and can also be performed along the z-axis in 3D space.


Translation Example

Formula

For 2D translation:
NewX = X + Tx
NewY = Y + Ty

For 3D translation:
NewX = X + Tx
NewY = Y + Ty
NewZ = Z + Tz

Steps for Translation

  1. Specify the translation distance along the x, y, and z axes (Tx, Ty, Tz).
  2. For each point (X, Y, Z) to be translated:
    1. Calculate the new X-coordinate: NewX = X + Tx
    2. Calculate the new Y-coordinate: NewY = Y + Ty
    3. Calculate the new Z-coordinate: NewZ = Z + Tz (for 3D translation)

Scaling

Scaling changes the size of an object in the graphic space. It can enlarge or shrink the object. Scaling can be uniform (in all dimensions) or non-uniform (in individual dimensions).


Scaling Example

Formula

For 2D scaling:
NewX = X * Sx
NewY = Y * Sy

For 3D scaling:
NewX = X * Sx
NewY = Y * Sy
NewZ = Z * Sz

Steps for Scaling

  1. Specify the scaling factors along the x, y, and z axes (Sx, Sy, Sz).
  2. For each point (X, Y, Z) to be scaled:
    1. Calculate the new X-coordinate: NewX = X * Sx
    2. Calculate the new Y-coordinate: NewY = Y * Sy
    3. Calculate the new Z-coordinate: NewZ = Z * Sz (for 3D scaling)

Rotation

Rotation transforms an object by an angle around a specified point or axis. In 2D graphics, rotation is typically around the origin (0,0). In 3D graphics, rotation can be around any arbitrary axis in 3D space.

Rotation Example


Formula

For 2D rotation:
NewX = X * cos(θ) - Y * sin(θ)
NewY = X * sin(θ) + Y * cos(θ)

For 3D rotation (around x-axis):
NewY = Y * cos(θ) - Z * sin(θ)
NewZ = Y * sin(θ) + Z * cos(θ)

Similar formulas apply for rotations around y-axis and z-axis in 3D space.

Steps for Rotation

  1. Specify the angle of rotation (θ) in radians.
  2. For each point (X, Y, Z) to be rotated:
    1. Calculate the new X-coordinate: NewX = X * cos(θ) - Y * sin(θ)
    2. Calculate the new Y-coordinate: NewY = X * sin(θ) + Y * cos(θ)
    3. Calculate the new Z-coordinate (for 3D rotation): NewZ = Y * sin(θ) + Z * cos(θ)
