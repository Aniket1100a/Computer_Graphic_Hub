Polygon Clipping: Sutherland-Hodgeman and Weiler-Atherton Algorithms

Sutherland-Hodgeman Algorithm

The Sutherland-Hodgeman algorithm is a polygon clipping algorithm used to clip a polygon against a rectangular window. It processes each edge of the polygon and clips it against the window boundaries.



Key Steps:

    For each polygon edge, determine if it is entering, exiting, or inside the window.

    Clip the entering edge and output the intersection point.

    For inside edges, output the intersection points.

    Clip the exiting edge and output the intersection point.

Advantages:

    Simple and easy to implement.

    Works well for convex polygons.

Limitations:

    Doesn't handle concave polygons properly.

    May produce extra vertices.

Weiler-Atherton Algorithm

The Weiler-Atherton algorithm is a more complex polygon clipping algorithm that handles polygons with holes. It generates the clipped polygon by computing intersection points and following contour paths.



Key Steps:

    Compute intersection points between the polygon edges and the window boundaries.

    Follow contour paths to create the clipped polygons.

    Handle cases where holes are involved in the clipping.

Advantages:

    Works for polygons with holes.

    Produces accurate results.

Limitations:

    More complex implementation compared to Sutherland-Hodgeman.

    Requires handling multiple intersection cases.
