Computer Graphics Units

Unit I: Graphics Primitives and Scan Conversion Algorithms

Description: This unit introduces fundamental concepts in computer graphics, including graphics primitives, pixel resolution, aspect ratio, frame buffer, display devices, and their applications.

Start Learning

Unit II: Polygon, Windowing and Clipping

Description: This unit covers polygons, polygon types (convex, concave, complex), polygon filling techniques (flood fill, seed fill, scan line fill), and windowing and clipping concepts.

Start Learning

Unit III: 2D, 3D Transformations and Projections

Description: This unit explores 2-D transformations (translation, scaling, rotation, shear), 3-D transformations (translation, scaling, rotation, shear, rotation about an arbitrary axis), and various types of projections (parallel and perspective).

Start Learning

Unit IV: Light, Colour, Shading and Hidden Surfaces

Description: This unit delves into color models, illumination models, shading algorithms, and hidden surfaces. Topics include RGB, HSV, CMY color models, Phong model, shading algorithms, and hidden surface removal.

Start Learning

Unit V: Curves and Fractals

Description: This unit covers curves, including interpolation, approximation, B-Spline, and Bezier curves, as well as fractals, their classification, and applications.

Start Learning

Unit VI: Introduction to Animation and Gaming

Description: This unit introduces animation concepts, animation languages, key-frames, morphing, motion specification, gaming platforms, and advances in gaming technology.
